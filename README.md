# Covid-19-data-analysis-project


  This project is involved statical interpretation with python pandas and seaborn. It involves
two datasets i.e, the COVID19 dataset, published by John Hopkins University and the other
dataset consist of various life factors, scored by the people living in each country around the
globe and then merge these two datasets to see if there is any relationship between the
spread of the virus in a country and how happy people are, living in that country
